# Product Overview

## Track Cricket Score App

A simple Android score keeper application for cricket matches. The app allows users to track scores and wickets for two teams during a cricket game.

### Key Features
- Track runs scored by each team (1, 2, 3, 4, 6 runs)
- Track wickets fallen for each team (max 10 wickets per team)
- Reset functionality to start new matches
- Simple, intuitive UI for real-time score tracking

### Target Audience
Cricket enthusiasts who want a basic digital scorecard for casual matches or practice sessions.

### Project Context
This is an educational project (ABND Project 2) focused on Android fundamentals including button interactions, view updates, variable scoping, and findViewById usage.
